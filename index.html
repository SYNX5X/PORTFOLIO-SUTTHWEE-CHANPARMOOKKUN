สุทธวีร์ ฉันท์ประมุขกุล

Sutthawee Chanparmookkun

สวัสดีครับท่านคณะกรรมการทุกท่าน ตามหัวข้อเลยครับ ผมชื่อ นายสุทธวีร์ ฉันท์ประมุขกุล ก่อนอื่นผมขอขอบคุณคณะกรรมการทุกท่านที่ได้เลือกและสละเวลามาอ่านเว็บไซต์ที่ผมได้จัดทำขึ้นในครั้งนี้ ซึ่งเป็นการนำสิ่งที่ผมได้เรียนรู้และค้นคว้าเกี่ยวกับการสร้างเว็บไซต์ด้วย HTML และ CSS มาใช้ครับ เว็บไซต์ HTML นี้มีทั้งหมด 6 หน้าด้วยกัน โดยผมจะเริ่มจากหน้านี้ก่อนเลย คือหน้า HOME ในหน้านี้จะมีข้อมูลเกี่ยวกับเหตุผลที่ผมเริ่มทำเว็บไซต์นี้ และเหตุผลที่ผมอยากศึกษาสายนี้ครับ เว็บไซต์นี้ผมได้จัดทำและดำเนินการบนโปรแกรม Visual Studio Code และใช้ Extensions - Live Server บน Visual Studio Code เพื่อให้สามารถเห็นเว็บไซต์ที่ทำได้แบบเรียลไทม์ครับ เว็บไซต์นี้เปรียบเสมือนการรวม Portfolio ของผมไว้ในที่เดียว และผมอยากเชิญให้คณะกรรมการได้ทดลองใช้และทดสอบเว็บไซต์ที่ผมทำขึ้นในครั้งนี้ครับ ในอนาคตผมอยากต่อยอดเว็บไซต์นี้ให้มีระบบ Database สำหรับการเก็บข้อมูลต่างๆ รวมถึงการพัฒนาในด้าน UX/UI เพื่อให้เว็บไซต์ใช้งานได้ง่ายและเป็นมิตรต่อผู้ใช้งานครับ ทั้งหมดนี้ผมได้นำความรู้ที่ได้รับจากการเรียนที่มหาวิทยาลัยเทคโนโลยีพระจอมเกล้าพระนครเหนือ คณะวิทยาศาสตร์ประยุกต์ สาขาวิทยาการคอมพิวเตอร์ มาประยุกต์ใช้ เพื่อสร้างสิ่งใหม่ๆ และสร้างสิ่งประดิษฐ์ที่มีคุณภาพให้กับผู้ใช้งานทุกคนครับ สุดท้ายนี้ คติประจำตัวของผมคือ “เมื่อเราพลาด เราควรรีบลุกขึ้น ปรับปรุง และแก้ไขข้อผิดพลาดของตนเองให้เร็วที่สุด เมื่อเราทำทุกอย่างเต็มที่แล้ว ผลลัพธ์ที่ดีจะตามมาแน่นอนครับ”

เหตุผลที่ผมถึงอยากเข้า คณะเทคโนโลยีสารสนเทศ มหาวิทยาลัยเทคโนโลยีพระจอมเกล้าธนบุรี

The Reason Why I Want to Join School of Information Technology Of King Mongkut's University of Technology Thonburi

เหตุผลที่ผมมุ่งมั่นจะเข้าศึกษาต่อในคณะเทคโนโลยีสารสนเทศ มหาวิทยาลัยเทคโนโลยีพระจอมเกล้าธนบุรี เกิดจากประสบการณ์ที่สร้างแรงบันดาลใจมาตั้งแต่สมัยมัธยมต้น ในช่วงที่เกิดการแพร่ระบาดของโควิด-19 ผมได้เรียนออนไลน์และมีโอกาสใช้คอมพิวเตอร์ส่วนตัวเป็นครั้งแรก ทำให้ผมได้สำรวจการใช้งานคอมพิวเตอร์อย่างลึกซึ้งและได้พัฒนาทักษะการใช้เทคโนโลยีไปพร้อม ๆ กับการเรียนรู้ด้วยตัวเอง ประสบการณ์นี้กลายเป็นพื้นฐานสำคัญที่ทำให้ผมหลงใหลและสนใจในสายเทคโนโลยีอย่างแท้จริง เมื่อผมได้ศึกษาต่อในระดับมัธยมปลายและย้ายโรงเรียนใหม่ ผมได้รับความรู้พื้นฐานทั้งในด้านซอฟต์แวร์และฮาร์ดแวร์ รวมถึงการพัฒนาเว็บแอปพลิเคชันเบื้องต้น สิ่งเหล่านี้ยิ่งทำให้ผมมุ่งมั่นที่จะต่อยอดความรู้ด้านเทคโนโลยีสารสนเทศ และทำให้ผมเชื่อมั่นว่าตนเองมีศักยภาพในการพัฒนาในสายอาชีพนี้ได้อย่างแน่นอน ความรู้พื้นฐานที่ได้รับทำให้ผมตั้งเป้าหมายที่จะเพิ่มพูนทักษะเพื่อสร้างความสำเร็จในสายวิชาชีพด้านเทคโนโลยี เมื่อผมได้พิจารณาคณะต่าง ๆ ที่สอดคล้องกับความสนใจ คณะเทคโนโลยีสารสนเทศของมหาวิทยาลัยเทคโนโลยีพระจอมเกล้าธนบุรีมีหลักสูตรที่ทันสมัยและครอบคลุมทักษะที่ผมต้องการพัฒนา ผมประทับใจทั้งในด้านการเรียนการสอนที่เน้นการประยุกต์ใช้งานจริง การเรียนรู้จากบุคลากรผู้เชี่ยวชาญ และสิ่งอำนวยความสะดวกที่ครบครันซึ่งจะช่วยให้ผมได้ฝึกฝนและเสริมประสบการณ์การเรียนรู้ได้อย่างเต็มที่ สภาพแวดล้อมที่เอื้อต่อการเรียนรู้และเทคโนโลยีที่ทันสมัยทำให้ผมมั่นใจว่ามหาวิทยาลัยแห่งนี้คือสถานที่ที่เหมาะสมที่สุดสำหรับการเติบโตในสายวิชาชีพที่ผมสนใจ ด้วยความตั้งใจนี้ ผมมุ่งหวังว่าการได้ศึกษาที่คณะเทคโนโลยีสารสนเทศ มหาวิทยาลัยเทคโนโลยีพระจอมเกล้าธนบุรี จะเป็นก้าวสำคัญที่ช่วยให้ผมพัฒนาทักษะและความรู้ให้ก้าวไกลยิ่งขึ้น เพื่อที่ผมจะได้นำความรู้และประสบการณ์ที่ได้รับไปสร้างประโยชน์ให้กับสังคม ทั้งในด้านการแก้ปัญหาต่าง ๆ ที่เกิดขึ้นในปัจจุบันและต่อยอดไปในอนาคต
