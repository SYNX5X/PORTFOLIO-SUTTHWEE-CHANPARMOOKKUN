สุทธวีร์ ฉันท์ประมุขกุล

Sutthawee Chanparmookkun

สวัสดีครับ ท่านคณะกรรมการทุกๆท่านนะครับตามหัวข้อเลยครับ ผมชื่อ นาย สุทธวีร์ ฉันท์ประมุขกุล อันดับเเรกผมขอบคุณคณะกรรมการทุกๆท่าน ที่ได้เลือก พอร์ตเเละได้เข้ามาอ่านเว็บไซต์ที่ผมได้จัดทำขึ้นในครั้งนี้เพื่อเป็นการนำสิ่งที่ได้เรียนรู้เเละค้นคว้าเรื่อว การทำเว็บไซต์ด้วย HTML เเละ CSS ครับ โดยตัวเว็บHTML นี้ก็จะมีทั้งหมด 6 หน้าด้วยกันนะครับ โดยผมจะเริ่มจากหน้านี้ก่อนเลย คือหน้า HOME นั้นเองนะครับ โดยในสวนนี้ก็จะมีทั้งข้อมูลว่าทำไมผมถึงได้เริ่มทำเว็บไซต์นี้ทำไมตัวผมเองนั้นถึงได้อยากศึกษาในสายทางนี้นั้นเองครับ โดยเว็บไซต์นี้ผมได้ จัดทำเเละดำเนินการบน Visual Studio Code ครับ ส่วนสิ่งที่ทำที่เป็นเว็บไซต์ได้เเบบเรียลทามนั้น ผมได้ใช้ จะเป็น Extensions บน Visual Studio Code คือ Live Server ครับเป็นตัวช่วยผมได้เห็นเว็บไซต์ที่ผมทำได้เป็นอย่างดีเลยละครับ เเล้วเว็บไซต์นี้เหมือนเป็นการนำ PORTFOLIO ของผมมาใส่ไว้ในเว็บไซต์นี้เว็บเดียวเลยครับ เเละต่อจากนี้ผมอยากให้คณะกรรมการได้ลองใช้หรือทดสอบเว็บไซต์ที่ผมได้ทำขึ้นในครั้งนี้นะครับ ซึ่งผมอยากต่อยอดเว็บไซต์ที่ผมทำนั้นให่มีเรื่องการทำ Data Base ในการเก็บของมูลต่างๆ หรือไม่ว่าจะะเป็นทั้งเรื่องการทำ UX/UI เพื่อทำให้หน้าตาของเว็บไซต์ใช้งานได้ง่ายเเละเป็นมิตรเเก่ผู้ใช้ทุกๆๆคนที่ได้เ้ามาใช้บริการเว็บไซต์ที่ผมต้องการที่จะทำนั้นเองครับ หรือการนำความรู้ที่ได้จากการเรียน มหาวิทยาลัยพระจอมเกล้าพระนครเหนือ คณะวิทยาศาสตร์ประยุกต์ สาขาวิทยาการคอมพิวเตอร์ เพื่อที่จะนำมาต่อยอดสิ่งใหม่ๆเเละสร้างสิ่งประดิษฐิ์ที่มีคุณภาพให้ผู้คนหรือประชาชนทุกคนได้ใช้งานอย่างมีประสิทธิภาพที่สุดครับ เเละคติประจำตัวของผม ความหมายของมันรงตัวมากๆครับ เมื่อเราพลาดทำสิ่งที่ผิดล้มขึ้นมา เราควรรีบลุกรีบปรับปรุงเเละเเก้ไขข้อผิดพลาดของตัวเองให้เร็วที่สุดเเล้วเมื่อเราทำทุกอย่างเเล้วผลมันก็จะออกมาดีเเน่นอนครับ

เหตุผลที่ผมถึงอยากเข้า คณะวิทยาศาสตร์ประยุกต์ สาขาวิทยาการคอมพิวเตอร์ ของมหาวิทยาลัยพระจอมเกล้าพระนครเหนือ

The Reason Why I Want to Join Faculty of Applied Science Science Program in Computer Science(CS) of King Mongkut's University of Technology North Bongkok

เหตุผลที่ผมอยากเข้า คณะวิทยาศาสตร์ประยุกต์ สาขาวิทยาการคอมพิวเตอร์ นั้น เมื่อผมยังอยู่มัธยมต้นช่วง ที่มีการเเพร่ระบาดของโควิด-19 นั้นทำให้ผมได้เรียนออนไลน์ซึ่งการเรียนออนไลน์ในครั้งนี้เป็นการเปิดประสบการณ์ทางด้านคอมพิวเตอร์ของผมเลยก็ว่าได้ครับเพราะผมได้คอมพิวเตอร์เป็นของตัวเองครั้งเเรกจึงทำให้ผมได้เรียนรูการใช้คอมพิวเตอร์ของผม ซึ่งด้วยในช่วงนั้นผมจึงมีเวลาอยู่กับหน้าจอคอมพิวเตอร์มากขึ้นได้ลองได้เรียนรู้การใช้คอมพิวเตอร์มากขึ้นจึงทำให้ผมสนใจเกี่ยวกับคอมพิวเตอร์ เเล้วเมื่อผมได้เรียนมาถึงมัธยมปลายผมได้ย้ายโรงเรียนผมจึงได้เรียนรู้ความรู็พื้นฐานไม่ว่าจะเกี่ยวกับ ทั้งท้านด้าน ซอฟต์เเวร์ เเละ ท้านด้าน ฮาร์ดเเวร์ ไม่ว่าจะเป็นการได้รู้พื้นฐานการทำเว็บเเอปพลิเคชัน บน เว็บไซต์ ซึ่งผมอยากต่อยอดความรู้ในสิ่งที่ผมได้เรียนรู้นี้ไปให้ดีที่สุดประสบความสำเร็จมากที่สุดเเละผมเหตุผลที่ผมอยากเข้า คณะวิทยาศาสตร์ประยุกต์ สาขาวิทยาการคอมพิวเตอร์ ของมหาวิทยาลัยพระจอมเกล้าพระนครเหนือ เพราะ มหาวิทยาลัยพระจอมเกล้าพระนครเหนือ นั้นมีอุปกรณ์ที่เพรียบพร้อมทั้งบุคลากรที่สอนนักศึกษา เรื่องเนื้อหาหลักศูตรการศึกษาที่ตอบโจทย์ผมเป็นอย่างมากครับไม่ว่าจะเป็นการสอนเรื่อง วิศวกรรมซอฟต์เเวร์ ระบบอัจฉริยะ ที่อยู่ตามมือถือสมัยนี้ต่างๆ ซึ่งเป็นสิ่งที่ผมสนใจเเละอยากเข้ามาศึกษาที่ มหาวิทยาลัยพระจอมเกล้าพระนครเหนือ เเละด้วยทั้งสถานที่การศึกษาที่เพรียบพร้อมเต็มไปด้วยสิ่งอำนวยความสะดวกต่างๆๆให้เเก่นักศึกษา เเละนี้คือเหตุผลที่ผมอยากเข้า คณะวิทยาศาสตร์ประยุกต์ สาขาวิทยาการคอมพิวเตอร์ ของมหาวิทยาลัยพระจอมเกล้าพระนครเหนือ นั้นเองครับ ขอบคุณครับ.
